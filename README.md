# Portal

### Scripts

```
# Comandos básicos (rodar na pasta raiz do projeto)
$ yarn dev # Inicia o servidor de desenvolvimento (HMR habilitado)
$ yarn build # Cria a versão de produção do aplicativo
$ yarn start # inicia o aplicativo de produção (da compilação)
$ yarn type-check # Verificações de tipo usando o typescript
$ yarn lint # Recursos de linting usando eslint

# Configura Project LF

$ git config core.autocrlf false
$ git rm --cached -r .
$ git reset --hard
```
